# raw-experience

Pi-пакет с набором расширений для pi-coding-agent.

## Расширения

### permission
Система разрешений — контролирует доступ к файлам и инструментам на основе правил (allow/deny). Защищает от случайного редактирования важных файлов.

### get-package-github-url
Утилита для получения GitHub-репозитория по имени npm/PyPI пакета.

### package-local
Управление локальными кэшами пакетов для офлайн-доступа к коду зависимостей. Поддерживает JavaScript/TypeScript и Python.

### subagent
Обобщённые субагенты с контролем максимальной вложенности. Каждый subagent запускается в отдельном pi-процессе с изолированным контекстом.

## Быстрый старт

Подключите пакет в `settings.json`:
```json
{
  "packages": [
    "path:/path/to/raw-experience"
  ]
}
```
