# raw-experience

Pi-пакет с набором расширений для pi-coding-agent.

## Расширения

### permission
Система разрешений — контролирует доступ к файлам и инструментам на основе правил (allow/deny). Защищает от случайного редактирования важных файлов.

### get-package-github-url
Утилита для получения GitHub-репозитория по имени npm/PyPI пакета.
